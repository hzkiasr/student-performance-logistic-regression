# Logistic Regression Analysis

## Project Overview

This project applies logistic regression techniques to analyze factors influencing student academic performance using survey-based data.

## Objectives

* Perform data preprocessing and cleaning.
* Transform categorical variables using encoding techniques.
* Conduct exploratory data analysis (EDA).
* Build logistic regression models.
* Evaluate relationships between independent variables and target outcomes.

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Statsmodels
* Scikit-learn

## Methods

* Data Cleaning
* Label Encoding
* Logistic Regression
* Statistical Testing
* Data Visualization
